WeMeet Help >> Groups

Use Groups link from the swipe menu to go to Groups screen.

This screen displays list of groups that you have creatd.

You can use add button on the top to create a new group.

Similarly, if you click on any group, it will open group details screen.
This screen contains list of group members.

You can use add button on the top to add a group member.

Use icon similar to map, to locate group members nearby.

You can long press any group or group member to delete it.
